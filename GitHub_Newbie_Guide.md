### GitHub 新手學習指南

GitHub 是程式碼版本控制和協作平台，以下是新手入門的關鍵步驟和資源：

#### 基本概念
- **Git vs. GitHub**: Git 是版本控制工具，GitHub 是線上平台，用來存放和分享 Git 倉庫。
- **倉庫 (Repository)**: 存放專案檔案的地方。
- **分支 (Branch)**: 允許平行開發，不影響主分支。
- **Pull Request**: 提出變更請求，讓他人審核。

#### 入門步驟
1. **註冊帳號**: 前往 [github.com](https://github.com) 註冊免費帳號。
2. **安裝 Git**: 下載並安裝 Git (從 [git-scm.com](https://git-scm.com))。
3. **建立倉庫**: 在 GitHub 上點擊 "New repository"，輸入名稱並初始化。
4. **克隆到本地**: 使用 `git clone <倉庫URL>` 將倉庫下載到電腦。
5. **基本命令**:
   - `git add .`: 加入檔案到暫存區。
   - `git commit -m "訊息"`: 提交變更。
   - `git push`: 上傳到 GitHub。
   - `git pull`: 從 GitHub 下載最新變更。

#### 學習資源
- **官方文檔**: [GitHub Docs](https://docs.github.com/zh) (有中文版)。
- **免費課程**: GitHub Learning Lab (在 GitHub 上搜尋 "learning lab") 或 [Codecademy Git 課程](https://www.codecademy.com/learn/learn-git)。
- **影片教程**: YouTube 搜尋 "GitHub for beginners"，推薦 The Net Ninja 或 Traversy Media 的影片。
- **實作練習**: 建立一個簡單專案，如個人網站，並練習推送/拉取。
